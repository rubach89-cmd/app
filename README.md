# Planteguide Ibestad (MVP)

Dette er en enkel Flutter-MVP for "Planteguide Ibestad" (lokal-only). Inneholder:
- Liste over 10 planter (norske tekster, assets/plants.json)
- Plant-detaljer
- Enkel dagbok per plante (tekst + bilde, lagret lokalt med SharedPreferences)
- Lokal-only (ingen backend)

Kjøring:
1. Installer Flutter (https://docs.flutter.dev/get-started/install)
2. Opprett nytt prosjekt eller bruk dette mappestrukturen.
3. Legg filene under `lib/` og `assets/` som vist.
4. I prosjektrot: kjør `flutter pub get`
5. Kjør appen: `flutter run`

Merk:
- For bildeopplasting bruker appen `image_picker`. På iOS/Android må du følge pakkenes installasjonsinstrukser (permissions i AndroidManifest/Info.plist).
- Dette er et startskjelett — dagboken lagrer entries lokalt med SharedPreferences (image paths lagres som streng).
